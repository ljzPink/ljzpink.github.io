开发工具

eclipse 快捷键

alt+/ 提示
格式化代码 : ctrl+shift+f
导入包： ctrl+shift+o
如果该类只有一个包，会直接导包
如果该类有多个包，会弹出对话框供选择
注释：单行注释：Ctrl+/ 取消再来一次
多行：Ctrl+shift+/
取消注释：Ctrl+shift+\
代码的上下移动：选中代码alt+↑/↓
查看源码：选中类名（F3或者Ctrl+鼠标点击）
创建对象：ctrl+2+L
选中要打印的变量 alt+/按两次 打印
删除当前行：ctrl+D
复制当前行：Ctrl+alt+↑
修改代码 alt+1
换行：shift +回车
到当行最前面/最后面：home/end
小写变大写：Ctrl+shift+x
大写变小写：Ctrl+shift+y
自动生成无参构造方法 alt+shift+s+c
自动生成有参构造方法 alt+shift+s+o
自动生成get set alt+shift+s+r
