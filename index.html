腾讯云搭建Java开发环境

前言：本次安装环境为腾讯云centos7，jdk1.8,tomcat9

安装JDK（使用yum安装）

第一步：查询yum源里面的资源

yum list java #yum search jdk

第二步：安装需要的JDK版本

yum install java-1.8.0-openjdk* -y

第三步：检查是否安装成功

java -version
结果为openjdk version “1.8.0_171”

安装tomcat

第一步：进入本地文件夹(文件夹可以自定义)

cd /usr/local

第二步：下载资源

到官网上找到下载资源，复制地址这里提供一个tomcat9的下载地址
wget http://www-eu.apache.org/dist/tomcat/tomcat-9/v9.0.10/bin/apache-tomcat-9.0.10.tar.gz

第三步：解压文件并重命名

解压：tar -zxf apache-tomcat-9.0.10.tar.gz
重命名：mv apache-tomcat-9.0.10 tomcat
给bin文件夹下所有的操作文件赋权限:
cd tomcat/bin/
chmod 777 *.sh

第四步：启动tomcat，并检查

./startup.sh
从浏览器打开ip:8080 弹出默认页面，完成

云服务器装MYSQL

前言：本人用的是腾讯云的服务器，系统为CentOS 7.4 64位

参考博客：https://blog.csdn.net/runner1920/article/details/79495368
具体步骤如下：
第一步：登陆云服务器（如果是腾讯云的话直接可以通过浏览器直接登陆，我还是比较习惯远程链接登陆，使用的是SecureCRT）
第二步：下载rpm文件,可在任意文件夹下下载
wget https://repo.mysql.com//mysql57-community-release-el7-11.noarch.rpm
第三步：安装rpm
rpm -ivh mysql57-community-release-el7-11.noarch.rpm
确认的时候输入y就好了，安装完成之后，在/etc/yum.repos.d目录下新增了两个文件，mysql-community.repo和mysql-community-source.repo（这个原文档里有，我没走执行这一步）
第四部：yum 安装mysql
yum install mysql-community-server
第五步：启动mysql服务
systemctl start mysqld
第六步：设置mysql开机启动
systemctl enable mysqld
systemctl daemon-reload
第七步：查找root密码 ，并登陆
mysql安装完成之后，在/var/log/mysqld.log文件中给root生成了一个默认密码。通过下面的方式找到root默认密码，然后登录mysql进行修改：
查看原始密码：grep 'temporary password' /var/log/mysqld.log
2018-06-28T03:31:47.372660Z 1 [Note] A temporary password is generated for root@localhost: 82%KWSxkK1o1
用原始密码 登陆:mysql -uroot -p82%KWSxkK1o1
修改密码：mysql> set password for 'root'@'localhost'=password('你要修改的密码')
注意：密码不能台简单不然会报错：
ERROR 1819 (HY000): Your password does not satisfy the current policy requirements
最好是有英文大小写，数字，特殊字符
第八步：修改/etc/my.cnf配置文件，在[mysqld]下添加编码配置，如下所示：
[mysqld]
character_set_server=utf8
init_connect='SET NAMES utf8'
第九步：重启mysql
systemctl restart mysqld

eclipse 快捷键

alt+/ 提示
格式化代码 : ctrl+shift+f
导入包： ctrl+shift+o
如果该类只有一个包，会直接导包
如果该类有多个包，会弹出对话框供选择
注释：单行注释：Ctrl+/ 取消再来一次
多行：Ctrl+shift+/
取消注释：Ctrl+shift+\
代码的上下移动：选中代码alt+↑/↓
查看源码：选中类名（F3或者Ctrl+鼠标点击）
创建对象：ctrl+2+L
选中要打印的变量 alt+/按两次 打印
删除当前行：ctrl+D
复制当前行：Ctrl+alt+↑
修改代码 alt+1
换行：shift +回车
到当行最前面/最后面：home/end
小写变大写：Ctrl+shift+x
大写变小写：Ctrl+shift+y
自动生成无参构造方法 alt+shift+s+c
自动生成有参构造方法 alt+shift+s+o
自动生成get set alt+shift+s+r
